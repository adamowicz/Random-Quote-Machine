# Random quote machine

You can use my "Random quote machine" to generate short (up to 140 signs) and smart quote and if you have an account on Twitter you can easily twitt it :-) 
Just push the button.

Used:

- HTML
- CSS
- RWD
- jQuery
- Ajax
- API
- Git
